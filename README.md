# Flask Microblog App 🚀

This is my first Flask project where I built a simple microblog web application.

## 🔥 Features

* Flask application setup
* Routing and view functions
* Jinja2 templates
* Dynamic user and posts display
* Template inheritance

## 🛠️ Tech Stack

* Python
* Flask
* HTML (Jinja Templates)

## 📁 Structure

microblog/
│
├── app/
│   ├── **init**.py
│   ├── routes.py
│   ├── templates/
│       ├── base.html
│       ├── index.html
│
├── microblog.py

## ▶️ Run Locally

1. Create virtual environment
2. Install Flask
3. Run:
   flask run

## 🌐 Output

http://127.0.0.1:5000/
